# Smart Recipe Generator

## Features

- Upload ingredient images
- Recognizes ingredients using Google Vision API
- Diet preferences: vegetarian, vegan, gluten-free
- Fetches recipes using Spoonacular API
- Step-by-step recipe instructions
- Nutrition information
- Shopping list generator (PDF + TXT)
- Modern clean UI with React + Tailwind

## Tech Stack

- Frontend: React + Tailwind
- Backend: Node.js + Express
- Image Recognition: Google Cloud Vision API
- Recipes: Spoonacular Free API
